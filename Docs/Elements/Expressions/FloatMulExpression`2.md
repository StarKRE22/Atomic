### üóÇ Example Usage

```csharp
var expression = new FloatMulExpression<float, float>(
    (a, b) => a,
    (a, b) => b,
    (a, b) => a + b
);
float result = expression.Invoke(2, 3); // 2 * 3 * (2 + 3) = 30
```
